# Robotic_Arm_Simulator

# Requirements
- Unity editor version: 6000.3.9f1
- Git
- Git LFS (for large assets)


# Setup Instructions
- Open git bash or cmd
- cd to where you want to clone the roep
- Type commands :
- git lfs install
- git clone https://github.com/UnrealizedEnd77/Robotic_Arm_Simulator.git
- cd Robotic_Arm_Simulator
- git lfs pull
- Then, open the project in unity hub
- In project window, in assets, open "arm"
  
# Controls
- Shoulder Joint :- left and right arrow
- shoulder :- up and down arrow
- arm :- 'R' and 'F' keys
- wrist :- 'T' and 'G' keys
- grippers :- 'Q' and 'E' keys
- Tanktreads :- WASD
